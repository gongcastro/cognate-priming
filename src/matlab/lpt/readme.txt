Acces LPT port from MatLab in Windos 64bits:
http://apps.usd.edu/coglab/psyc770/IO64.html

* copy "inpoutx64.dll" in c:\Windows\System32
* copy "io64.mexw64" in a folder accesible by Matlab Path
* use "lptx64.m" as a sample


frist time run mathlab as administrator
[Note: Self-installation of the driver requires that the MATLAB runs with Administrator privileges.  The driver must have been previously installed in order to support non-Administrator users].
